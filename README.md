# 🚀 AI-Powered Task Management System

A backend system similar to Jira with AI-powered features.

## 🔧 Tech Stack
- Java Spring Boot
- Spring Security + JWT
- PostgreSQL (Docker)
- JPA / Hibernate

## 🔐 Features
- User Authentication (JWT)
- Role-Based Access Control (ADMIN, MANAGER, USER, DEVELOPER)
- Task Management (Create, Assign, Update Status)
- Secure APIs

## 🧠 Upcoming Features
- AI Task Description Generator
- Smart Priority Detection
- Chat-based Task Queries

## 🚀 How to Run
1. Start PostgreSQL (Docker)
2. Run Spring Boot app
3. Use Postman to test APIs
